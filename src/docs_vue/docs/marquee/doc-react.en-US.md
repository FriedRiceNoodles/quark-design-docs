# Marquee

### Intro

Text scrolling.

### Install

```tsx
import { Marquee } from "@quarkd/quark-react";
```

### Basic Usage

```html
<marquee title="{title}"></marquee>
```

### Speed

```html
<marquee title="{title}" speed="25"></marquee>
```

## API

### Props

| Attribute | Description                  | Type     | Default |
| --------- | ---------------------------- | -------- | ------- |
| title     | Title                        | `string` |
| speed     | Whether to hide left content | `string` | `50`    |
